들여쓰기 문단 구조
1. 오랜된 어플 갱신 조건
- 갱신 실패시 : 오류횟수 증가 (MOD_ERR_CNT + 1)
- 갱신 성공시 : 오류횟수 초기화 (MOD_ERR_CNT = 0)
- 갱신 대상 : 최종 수집일자가 10이상 되었고 갱신 실패 횟수가 10회 미만인 어플
2. Google Analytics (구글 방문통계)
- 계정 (ceo@dnt7.com / dntechdams)
3. App's Today
- 최근3일 집계 재생성 : /todayStats.do
- Price 그래프 재생성 : /todayContent.do?conType=Price&refresh=Y
- Category 그래프 재생성 : /todayContent.do?conType=Category&refresh=Y
- Rating 그래프 재생성 : /todayContent.do?conType=Rating&refresh=Y
- Review 그래프 재생성 : /todayContent.do?conType=Review&refresh=Y
4. 프런트 좌/우측 컨텐츠
- 프런트의 좌/우측 컨텐츠는 사이트의 성능을 고려하여 메모리에 데이터를 올려놓고 사용한다.
- 좌측 컨텐츠와 우측 배너 영역의 데이터를 초기화 하고 다시 생성려면 아래 링크를 클릭하세요.
[컨텐츠 갱신]
5. 프런트 미니차트
- 미국, 한국, 기타 국가 순서의 기준으로 출력된다. 미국 이나 한국의 랭킹에 없을 경우 기타 국가 순으로 4개국이 출력된다.
- 각 국가의 랭킹 조회 순서는 Top Overall(전체)에 랭킹이 있는 경우 Top Overall의 랭킹이 출력된다.
6. AppChart 상세 페이지의 지정된 일자의 그래프 보기
- ex) http://appchart.net/appContentGo.do?appcode=343200656&nation=143441&findDate=20100605
appcode : 어플코드
nation : 국가코드
findDate : 그래프의 끝일자
7. Query
- CATEGORY='25204'-- 전체 카테고리
- Paid(30), Free(27), Grossing(38)
- STR_TO_DATE(#reg_date#, '%Y-%m-%d')
- DATE_FORMAT(A.REG_DATE,'%Y-%m-%d')
